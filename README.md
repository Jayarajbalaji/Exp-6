# Exp - 6 STUDENT ONLINE EXAM REGISTRATION SYSTEM
# AIM:

To develop a system that allows students to register online for examinations, enabling exam authorities and administrators to manage exam schedules, registrations, and results efficiently.

# SRS (Procedure):
# 1. Student Registration

Student creates an account by entering personal details.

System verifies email/phone and activates the account.

# 2. Login & Profile Management

Student logs in and updates personal information if needed.

# 3. Exam Creation (Admin / Exam Authority)

Admin creates exams, sets exam code, title, date, time, venue, and maximum seats.

Exam is published for student registrations.

# 4. Exam Registration by Student

Student views available exams.

Selects exam and submits registration form.

Payment is completed if required.

System confirms registration and generates admit card.

# 5. Attendance & Exam Conduct

Student presents admit card.

Invigilator marks attendance.

Exam is conducted.

# 6. Marks Entry & Result Publication

Examiner uploads marks.
